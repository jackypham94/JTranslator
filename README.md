## CHỨC NĂNG CHÍNH:
- Quét chọn text để dịch
- Double click để dịch
- Kết quả từ Mazii và Google Translate
- Hỗ trợ cách đọc hán tự
- Hỗ trợ tra Kanji chi tiết & ví dụ
- Dịch từ tiếng Nhật sang Việt và ngược lại
- Morden UI
- Open source
